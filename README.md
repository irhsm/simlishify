# simlishify

A Svelte app that transforms your lyrics into playful, nonsensical Simlish!

## Run the app

```bash
# Install dependencies
npm i

# Run the app
npm run dev
```

## What’s the Idea?

So, when tools like SUNO AI try to sing in English (or any human language really), they often sound a bit off — because AI doesn’t feel emotion the way humans do.
The result? Uncanny-valley karaoke.

That’s where **Simlishify** comes in.

It takes your text — English, French, Indonesian, whatever — and **shuffles it into expressive nonsense,** a.k.a. Simlish-style gibberish.

Now, when you feed that into SUNO AI (or any voice model), it suddenly sounds way better — because nobody knows what it means. The pressure’s off, and the vibe’s on. 🎵

## Fun fact

Simlish (the nonsense language from The Sims) was created to avoid localization headaches and emotional weirdness — exactly what we’re doing here, but for AI music.
